position

static : 일반적인 적층 개념 - 작성하지 않아도 기본값
relative: 상대적인 좌표값을 설정 absolute : 선언된 영역을 절대좌표의 위치에 배치 가능 fixed : 현재 보여지는 브라우저의 공간을 기준으로 고정시킴

css에 z-index 값(정수)을 통해 전면부나 후면부로 배치 가능하다.
